Two Sum

Solution Steps

  • Iterate through the array while storing numbers in a hash map.
  • For each number, check if target - num exists in the map.
  • If found, return indices of the two numbers.
  • Otherwise, insert the number into the map and continue.

Solution Code (C++)


class Solution {
public:
    vector twoSum(vector& nums, int target) {
        unordered_map mp;
        for (int i = 0; i < nums.size(); i++) {
            int complement = target - nums[i];
            if (mp.count(complement)) {
                return {mp[complement], i};
            }
            mp[nums[i]] = i;
        }
        return {};
    }
};
