Overview

Obesity affects over 40.3% of US adults (CDC, 2021-2023), with medical costs exceeding $173 billion annually. This project analyzes a UCI dataset (2,111 individuals from Mexico, Peru, Colombia) with 16 features (age, height, weight, diet habits like vegetable consumption, physical activity) to perform unsupervised clustering for obesity risk.

Goal: Group individuals into distinct clusters based on lifestyle and health attributes aligning with known obesity risk categories using K-Means and Hierarchical Clustering.

Tech Stack

  • R (kmeans, hclust for clustering, silhouette for validation)
  • Data: Full dataset (no preprocessing needed), scaling for Euclidean distance
  • Visualization: Histograms, bar charts, scatterplot matrices, dendrograms
  • Tools: Gap statistic, elbow method, silhouette coefficient for optimal K

Key Features & Analysis

  • Data Handling: 17 variables (16 features + obesity class). Mix of categorical (e.g., gender, smoking) & numerical (e.g., age 14-61, weight 39-173kg, FCVC vegetable freq., FAF activity hours).
  • EDA Insights: Near-equal gender (slightly more males); mostly non-smokers. Weight right-skewed (peaks 65-70kg, 90-100kg); FCVC bimodal (moderate-high veggie intake); Age young adults (20-25 peak); FAF peaks at 0-2 hours (many inactive).
  • Clustering Prep: Converted categoricals to numeric; scaled all features for equal weighting.
  • Height-Weight: Positive linear relationship; Age-Weight slight positive trend.

Results

Optimal Clusters: K=10 selected via gap statistic, elbow method, and silhouette analysis for balanced, cohesive groups.

Model Comparison: K-Means faster with controlled K; Hierarchical provides dendrograms but slower for large data. Validated internally (silhouette coefficient), externally (post-hoc labels), and relatively (gap stats)‚ÄîK-Means showed stronger performance.

Clusters reveal patterns in diet/activity aligning with obesity risks (e.g., low FAF + high weight in high-risk groups). Embed plots here (e.g., silhouette plot, K-Means visualization PNG from R).
