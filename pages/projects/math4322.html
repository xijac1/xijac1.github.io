Overview

Heart disease is the leading cause of death in the US (CDC, 2022), claiming a life every 33 seconds. This project uses a UCI dataset (303 patients from Cleveland, Hungary, Switzerland, VA Long Beach) with 13 features (age, sex, chest pain type, etc.) to build classification models predicting heart disease presence (binary: yes/no).

Goals: Identify key predictors, compare models (Logistic Regression vs. Random Forest), and answer: Which features matter most? What's the best model? Does age matter? Is it more common in males?

Built by Group 5: Almin Raja, Dinh Mai, Julio Amaya, Subhan Mehmood, Nicholas Pham.

Tech Stack

  • R (base, glm for Logistic Regression, randomForest package)
  • Data: CSV import via read.csv(), cross-validation (80/20 splits)
  • Visualization: varImpPlot for feature importance, confusion matrices
  • Tools: Quarto for report generation, UCI ML Repository dataset

Key Features & Analysis

  • Data Handling: 14 variables (13 predictors + binary response 'num'). Categorical (e.g., sex, cp) & numerical (e.g., age, chol) features.
  • Logistic Regression: Optimal model with 8 significant predictors (sex, cp, trestbps, thalach, exang, slope, ca, thal). 82.46% accuracy via 10-fold CV.
  • Random Forest: 500 trees, handles non-linearity; 80.33% accuracy (full model). ca, thal, cp as top features.
  • Insights: Age not significant; heart disease 55% more likely in males (vs. 26% females).

Results

Best Model: Logistic Regression outperformed Random Forest (17.54% vs. 19.67% error rate). Lowest AIC/residual deviance with key predictors.

Top Predictors: Chest pain type (cp), vessels (ca), thalassemia (thal), sex.

Full report metrics: Confusion matrices, p-values, and step() AIC selection confirm robust predictions for medical triage.
